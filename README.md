# Muhammad Rizky Project-Based Cost Estimator

A professional web-based HPP (Harga Pokok Produksi / Cost of Goods Manufactured) calculator designed specifically for furniture and custom manufacturing businesses.

## üöÄ Features

### Multi-Material Type Support
- **Panel Materials** - Calculate area-based materials (plywood, MDF, etc.) with waste allowance
- **Linear Materials** - Calculate length-based materials (edging, molding, etc.) with waste allowance
- **Liquid Materials** - Volume-based calculations (paint, glue, etc.) with waste allowance
- **Unit Materials** - Discrete items (hinges, handles, etc.)
- **Fabric Materials** - Textile calculations with waste allowance

### Advanced Calculation Features
- **Multi-item Hardware Tracking** - Add multiple hardware items with detailed breakdown
- **Multi-item Labor Cost** - Track different labor operations (cutting, assembly, finishing, etc.)
- **Automatic Waste Calculation** - Built-in waste allowance for materials
- **Per-unit Cost Options** - Toggle costs between per-project and per-unit
- **Overhead Calculation** - Percentage-based overhead on subtotal
- **Profit Margin** - Configurable profit percentage

### User Experience
- **Auto-scroll to New Cards** - Automatically scroll to newly added material cards
- **Click-based Dropdown Menus** - Easy-to-use material type selector
- **Professional Grey Theme** - Calm, easy-to-read color scheme
- **Responsive Design** - Works on desktop, tablet, and mobile devices
- **Print Optimization** - Clean, single-page HPP Summary for printing

### Summary & Reporting
- **Detailed Material Breakdown** - See all materials with calculations
- **Hardware Detail** - Individual hardware items listed
- **Labor Detail** - Individual labor operations listed
- **Waste Tracking** - Shows waste allowance for each material
- **Professional Summary** - Complete cost breakdown with totals

## üì¶ Installation

1. Clone the repository:
```bash
git clone https://github.com/mrizkymxx/Muhammad-Rizky-Project-Based-Cost-Estimator.git
cd Muhammad-Rizky-Project-Based-Cost-Estimator
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open your browser and navigate to the URL shown in the terminal (usually `http://localhost:5173`)

## üöÄ Build for Production

```bash
npm run build
```

The production-ready files will be in the `dist` folder.

## üí° How to Use

1. **Enter Project Details**
   - Set project name
   - Set number of units to produce

2. **Add Materials**
   - Click "Add Material" button
   - Choose material type (Panel, Linear, Liquid, Unit, or Fabric)
   - Fill in material details and calculations
   - Material cards support auto-calculation and waste tracking

3. **Add Hardware**
   - Click "Add Hardware" button
   - Enter hardware name, quantity, and price
   - Toggle "per unit" if the cost applies to each unit

4. **Add Labor Costs**
   - Click "Add Labor" button
   - Enter labor operation name and cost
   - Toggle "per unit" if the cost applies to each unit

5. **Set Overhead & Profit**
   - Enter overhead percentage
   - Enter desired profit margin percentage

6. **View Summary**
   - Scroll to HPP Summary section
   - See complete cost breakdown
   - Print for client presentation
